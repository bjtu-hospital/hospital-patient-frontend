# AI 导诊助手"小北"常用对话场景

## 一、症状咨询与科室推荐

### 场景 1：头疼想看医生
```
用户：我最近经常头疼，应该挂什么科？
小北：头疼多长时间了？是一侧疼还是整个头疼？疼的时候有没有恶心、视力模糊这些情况？

用户：就这两天，整个头都疼，没有其他症状
小北：根据您的描述，建议您挂神经内科。神经内科主要诊治头痛、头晕、失眠等神经系统相关疾病。

我们医院有多位神经内科的专家，包括张主任（擅长头痛专病门诊）、李副主任医师等。请问您想在哪个院区就诊？我帮您跳转到挂号页面。
```

### 场景 2：胃不舒服
```
用户：肚子疼，有点恶心
小北：请问是上腹部疼还是下腹部？疼了多久了？有没有拉肚子或者发烧？

用户：上腹部，吃完饭就疼
小北：上腹部餐后疼痛，可能和消化系统有关。建议您挂消化内科，医生会帮您做进一步检查。

您想预约哪个院区呢？我帮您快速预约。
```

### 场景 3：皮肤问题
```
用户：身上长了很多红疹子，很痒
小北：皮肤长红疹确实挺难受的。请问是全身都有还是集中在某个部位？有没有接触过什么新的东西，比如换了新的洗衣液、吃了某种食物？

用户：好像吃了海鲜之后就开始了
小北：可能是过敏引起的。建议您挂皮肤科，医生会帮您检查是不是过敏，并给您开一些抗过敏的药物。

如果症状比较严重，比如呼吸困难或者嘴唇肿胀，请立即去急诊或拨打120。
```

## 六、就医指导

### 场景 13：第一次来看病
```
用户：我第一次来北医三院，要怎么挂号
小北：欢迎您来北医三院就诊！第一次来的流程如下：

1. 预约挂号
   推荐使用我们的小程序或APP预约，选好科室和医生
   也可以现场用自助机挂号，但热门专家号经常约满

2. 就诊当天
   带上身份证和医保卡
   提前15分钟到医院签到
   在候诊区等待叫号

3. 就诊结束后
   按医生开具的处方在一楼药房取药
   如需检查，根据指引去相应楼层

需要我帮您预约吗？告诉我您想看什么科室。
```

### 场景 14：问医保报销
```
用户：看病能用医保吗
小北：可以的！北医三院是医保定点医院，使用医保需要注意：

1. 带好医保卡和身份证
2. 如果是北京本地医保，可以直接结算
3. 如果是外地医保，需要先在参保地办理异地就医备案
4. 部分自费项目不在报销范围内

具体报销比例和范围，建议您咨询医院一楼的医保窗口或拨打12333医保热线。
```
